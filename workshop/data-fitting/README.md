# Data Fitting & Optimization

- **data_fitting_general.ipynb**: Showcases several different methods of fitting models to data: `numpy.polyfit`, `scipy.optimize.curve_fit`, and `scipy.odr`.
- **data_fitting_hubble.ipynb**: Contains most of the same content as `data_fitting_general.ipynb`, but demonstrates the use of `numpy.polyfit` in an astrophysical example based on Hubble's Law.
